# Web application using Spring Boot


## Description

This web application depicts Create, read, update and delete operations.I have used Thymeleaf for frontend,Spring Boot for backend & MySql as database.

### Tech Stacks

* Java
* Spring Boot
* Spring Framework
* Marven
* MySql
* Thymeleaf
